Введение.

Данная программа представляет собой телеграм-бота, который подсчитывает расходы и доходы. Его особенность заключается в том, что он уже содержит в себе базу данных и файл с настроенными запросами.
Для того, чтобы из этого решения сделать таск-менеджер, нужно:

1) настроить пользовательские действия в personal_actions.py, 
2) запросы в базу данных в db.py,
3) настроить структуру базы данных в SQLiteStudio.


Запуск. 

1) Создать проект в Pycharm.
2) Скопировать файлы из Github в директорию проекта.
3) Скачать aiogram:
pip install aiogram 
либо же через настройки проекта.
4) Запустить bot.py.
5) Начать говорить с телеграм-ботом @SuccessfulSuccessBot

Доступные команды написаны в personal_actions.py:

/start - начать
/spent 100 - сказать боту, что потратили 100 рублей. Можно писать любую сумму.
/earned 1000 - сказать боту, что заработали 1000 рублей. Можно писать любую сумму.
/history day/month/year - история затрат за день или месяц или год.


Дополнительные сведения. 


aiogram использует методологию асинхронного программирования, поэтому там возникли невиданные доселе await и async.
